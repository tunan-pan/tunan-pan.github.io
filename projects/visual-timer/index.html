Custom Visual Timer

sandbox  |  Web/UX
Try the timer →

My first "vibe coding" project with Claude. I created initial mockups in Figma, described my vision, and iteratively refined the code (both by hand and through Claude).

This came about as I needed a 4 minute timer for a board game, but existing visual timers only offered preset durations — usually an hour. I didn't want to see 4 minutes as a tiny sliver of a 60-minute circle, I wanted the full circle to represent my chosen time. So I built my own customizable visual timer where you set the max time, and the entire circle represents exactly that.
