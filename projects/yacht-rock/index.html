Yacht Rock — Making an Accessible Game on Board Game Arena [WIP]

My Role:

  • UX Designer
  • Front-End Contributor

Deliverables:

  • Adapted graphic assets
  • High-fidelity mockups/prototypes
  • Gameplay interaction flow diagram
  • Game on the BGA platform
  • Accessibility document

This is a collaborative project with my partner, to bring “Yacht Rock” to Board Game Arena (BGA), an online platform for playing digital versions of physical board games. What began as a small game development project became an opportunity to address a larger problem: accessibility on BGA and in Javascript.

While WCAG guidelines exist broadly for web accessibility, there is no practical guidance specific to Javascript, let alone interfaces within the BGA ecosystem. So as we continue to develop Yacht Rock, I have been collecting my learnings into a living accessibility document, in the hopes of helping other BGA developers incorporate accessibility in future BGA games.
